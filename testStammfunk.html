Trainieren Sie das Bestimmen von Stammfunktionen

Zur Eingabe (ist vom Browser abängig): Falls der Exponent kleiner als neun ist, z.B. drei so geben Sie einfach x^3 ein, das ^–Zeichen wird eventuell angezeigt.
Bei Exponenten größer als neun müssen Sie eventuell zweimal die Taste ^ drücken, umd das ^-Zeichen zu bekommen.
Beispiel für den Exponenten 10: Eingabe x^10
Ein "Malpunkt" / · wird mit dem * eingeben oder man läßt ihn weg. Also z.B. 42x oder auch 42*x.
Bei Aufgaben mit Parametern ist es aktuell nicht möglich einen durchaus richtigen Bruch wie z.B. c/2 einzugeben, falls ich Zeit finde korrigiere ich es noch. Statt dessen muss 1/2 c eingegeben werden.

Danach klicken Sie auf check. Der Computer korrigiert ggf. die Schreibweise Ihrer Eingabe. Sollte das nicht klappen, so steht ein NaN im Formularfeld. Danach prüft er, ob Sie eine richtige Stammfunktion angegeben haben.
